Возможности расширения

🎯

Автоловля поставок

Автоматический поиск и бронирование поставок по заданным критериям. Фильтры по дате и коэффициенту приемки. Работает в цикле до успеха.

🔄

Перераспределение остатков

Автоматическое перемещение товаров между складами. Выбор складов, ввод артикула и количества. Полная автоматизация процесса.

📅

Умные фильтры

Выбор конкретной даты или диапазона. Фильтрация по коэффициенту от бесплатно до 20x. Автоматический пропуск недоступных дат.

🏭

Мониторинг через API

Получение списка всех доступных складов и дат через официальный WB API. Реальные данные в реальном времени.

⛔

Защита от ошибок

Автоматический пропуск "Пока недоступно". Циклическая работа до успеха. Подробное логирование всех действий.

💬

Telegram уведомления

Интеграция с Telegram для получения уведомлений о успешных бронированиях прямо в мессенджер.

Быстрый старт

1

Получите API токен

Войдите в личный кабинет Wildberries → Настройки → API → Создайте новый токен с правами на просмотр поставок

2

Настройте расширение

Кликните на иконку расширения → Настройки → Введите API токен. При желании настройте Telegram для уведомлений.

3

Выберите функцию

Автоловля поставок - для автоматического бронирования. Перераспределение - для перемещения товаров между складами.

4

Запустите автоматизацию

Настройте фильтры, включите тумблер и наблюдайте за работой в консоли (F12). Расширение будет работать до успеха!

💡 Совет: Откройте консоль браузера (F12) для просмотра подробных логов работы расширения. Все действия логируются с эмодзи для удобства!
